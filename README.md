## 📌 Overview

PulseERP Operations Suite is a modern enterprise dashboard designed to streamline workforce operations, employee management, attendance tracking, departmental monitoring, reporting, and task workflows.

The platform provides business-oriented analytics and operational insights through a responsive and scalable Angular architecture.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/pulseerp-operations-suite.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

Open:

```bash
http://localhost:4200
```
